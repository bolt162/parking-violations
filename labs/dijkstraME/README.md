## Dijkstra Mutual Exclusion 

The following code shows how to implement Dijkstra's ME 
(1965) algorithm. One of the first multi-user (thread, 
n > 2) implementations.

Note there are better (newer) algorithms available that 
take into account fairness (equal access) to the critical 
section. Nevertheless, DijkstraME approach illistrates
the common features of ME algorithms.
